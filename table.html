Table

That the table tag opens a table. Each row is defined by a tr Tag. Each col within a row is defined by td tag. The th tag is used fpr data headers

There are lots of attributes that table can use. Some examples are below

  • border - this sets a border fpr the table tag.
  • width - sets the width of a table or column
  • height - sets the height of a table or column
